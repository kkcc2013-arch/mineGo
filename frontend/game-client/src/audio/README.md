# 音频资源清单

## 音效（Sound Effects）

### UI 音效
- **ui_click.mp3** - 按钮点击音效（时长：0.1s）
  - 用途：所有按钮、卡片点击
  - 特点：轻快、清脆
  
- **ui_back.mp3** - 返回操作音效（时长：0.15s）
  - 用途：返回按钮、关闭弹窗
  - 特点：柔和、向后感
  
- **ui_notification.mp3** - Toast 弹出音效（时长：0.3s）
  - 用途：提示消息、通知
  - 特点：提示性、温和
  
- **ui_error.mp3** - 错误提示音效（时长：0.25s）
  - 用途：操作失败、验证错误
  - 特点：警示、但不刺耳

## 技术规范

### 格式要求
- **格式**：MP3（MPEG Audio Layer 3）
- **采样率**：44100 Hz
- **比特率**：128-192 kbps
- **声道**：立体声（音乐）、单声道（音效）

### 文件大小限制
- **音效**：< 100 KB（单个文件）
- **背景音乐**：< 500 KB（单个文件，压缩后）

### 音量规范
- **音效峰值**：-3 dB
- **背景音乐峰值**：-6 dB（避免喧宾夺主）
- **RMS 音量**：-18 dB

### 命名规范
- **格式**：小写字母、下划线分隔
- **示例**：ui_click.mp3, catch_caught.mp3
