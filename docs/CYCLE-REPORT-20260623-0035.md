## 技术亮点

### 1. 零停机轮换策略

```
旧版本 active → deprecated（24小时过渡期）
新版本 active
→ 旧版本 revoked
```

确保正在进行的请求可以继续使用旧密钥验证，新请求使用新密钥。

### 2. 多层安全防护

- **存储层**：AES-256-GCM 加密 + GCM 认证标签防篡改
- **访问层**：5 分钟缓存 + 审计日志
- **日志层**：敏感字段自动脱敏
- **应急层**：1 分钟内紧急撤销/轮换

### 3. 降级机制

KMS 不可用时自动降级到环境变量，确保服务可用性。
