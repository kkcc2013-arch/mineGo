| 属性 | 值 |
|------|-----|
| 状态变更 | new → done |
| 实现时间 | ~30 分钟 |

#### 实现内容

**数据库迁移**：
- `status_effect_definitions` - 状态效果定义表
- `status_effect_mechanics` - 状态机制表
- `battle_pokemon_status` - 战斗中精灵状态表
- `battle_stat_changes` - 能力变化记录表
- `type_status_immunities` - 属性免疫表
- `ability_status_immunities` - 特性免疫表

**状态效果引擎**：
- 45+ 种状态效果定义
- 完整的施加/移除/驱散逻辑
- 免疫检查系统
- 能力变化计算（±6级）
- 回合开始/结束处理

**API端点**（12个）：
- GET `/definitions` - 获取状态效果定义
- GET `/:battleId/:pokemonId` - 获取精灵状态
- POST `/apply` - 施加状态效果
- POST `/remove` - 移除状态效果
- POST `/dispel` - 驱散状态效果
- POST `/check-action` - 检查行动阻止
- POST `/turn-start` - 处理回合开始
- POST `/turn-end` - 处理回合结束
- GET `/field/:battleId` - 获取场地效果
- POST `/stat-change` - 应用能力变化
- POST `/calculate-stats` - 计算修正属性
- DELETE `/battle/:battleId` - 清除战斗状态

#### 状态效果类型覆盖

| 类别 | 数量 | 示例 |
|------|------|------|
| 控制类 | 10 | 灼伤、麻痹、冰冻、睡眠、混乱 |
| 持续伤害类 | 5 | 中毒、剧毒、寄生种子 |
| 能力变化类 | 13 | 攻击提升/下降、速度提升/下降 |
| 场地效果 | 8 | 大晴天、求雨、沙尘暴 |
| 防御/特殊 | 9 | 守住、替身、蓄力 |
