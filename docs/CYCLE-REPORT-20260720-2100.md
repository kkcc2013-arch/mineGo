## 任务 1: 生成新需求 - REQ-00620

### 需求信息
- **编号**：REQ-00620
- **标题**：游戏离线体验与 Service Worker 智能缓存管理系统
- **类别**：前端体验
- **优先级**：P1
- **状态**：new

### 需求背景
移动场景下网络不稳定是常态，当前游戏缺乏离线体验，导致：
- 用户流失率高
- 加载时间长
- 流量消耗大
- 离线场景无法留存用户

### 核心功能
1. **Service Worker 缓存策略**：Cache-First、Network-First、Stale-While-Revalidate
2. **离线数据管理**：IndexedDB 存储、离线操作队列、网络恢复同步
3. **网络状态监控**：在线/离线检测、智能重连、降级 UI
4. **性能优化**：首屏加载时间减少 50%，流量消耗减少 60%

### 验收标准
- ✅ 实现 5 种缓存策略
- ✅ 预缓存 50 个核心资源
- ✅ 离线查看精灵、背包、成就
- ✅ 网络恢复自动同步
- ✅ 缓存命中率 >= 80%
