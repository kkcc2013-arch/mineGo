## 任务详情

### 1. 生成新需求

| 项目 | 内容 |
|------|------|
| 编号 | REQ-00188 |
| 标题 | 用户语言偏好持久化与跨设备同步系统 |
| 类别 | 国际化/本地化（轮转索引 188 % 18 = 10） |
| 优先级 | P1 |
| 状态 | new |

**需求背景**：
- 语言偏好仅存储在浏览器 localStorage，更换设备后设置丢失
- 无跨设备同步，用户体验不一致
- users 表缺少 language_preference 字段
- 服务端中间件已支持但数据库字段缺失

**核心功能**：
- 数据库添加 language_preference 字段
- 语言偏好设置 API
- 登录后自动同步语言偏好
- 语言变更审计日志

**装备类型体系**：
| 类型 | 说明 | 属性加成 |
|------|------|----------|
| weapon | 武器 | 攻击、暴击率、暴击伤害 |
| armor | 护甲 | 防御、生命、抗性 |
| accessory | 饰品 | 速度、闪避、特殊属性 |
| skill_disc | 技能盘 | 技能伤害、技能冷却 |
| evolution_stone | 进化石 | 进化加速、经验加成 |
| held_item | 携带道具 | 特殊效果、被动能力 |

**稀有度系统**：
| 稀有度 | 掉率 | 最大等级 | 属性系数 |
|--------|------|----------|----------|
| common | 50% | 5 | 1.0x |
| uncommon | 30% | 7 | 1.2x |
| rare | 15% | 10 | 1.5x |
| epic | 4% | 12 | 2.0x |
| legendary | 1% | 15 | 3.0x |

**API 端点**：
- `GET /equipment/templates` - 获取装备模板列表
- `GET /equipment/inventory` - 获取玩家装备背包
- `POST /equipment/equip` - 装备到精灵
- `POST /equipment/unequip` - 从精灵卸下装备
- `POST /equipment/upgrade` - 强化装备
- `GET /equipment/sets` - 获取套装列表
- `GET /equipment/pokemon/:id` - 获取精灵已装备列表
- `POST /equipment/sell` - 出售装备

**技术亮点**：
- 元素亲和限制：水系精灵只能装备水系或通用装备
- 套装效果系统：2件/4件/6件套装效果
- 强化系统：消耗计算、成功率、失败处理
- 战斗属性计算：基础 + 装备 + 套装效果
- 自动卸下旧装备：装备同类型时自动替换
