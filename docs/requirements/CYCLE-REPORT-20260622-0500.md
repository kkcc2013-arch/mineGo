1. **一键环境初始化** (`npm run setup`)

   - Docker 服务启动

   - 种子数据填充

2. **环境健康检查** (`npm run doctor`)
   - 10+ 检查项
   - 每项提供修复建议
   - 自动诊断覆盖率 ≥ 95%

3. **快速重置** (`npm run reset -- --full`)
   - 5 分钟内完成环境重置
   - 支持部分重置选项

4. **VS Code 集成**
   - 推荐扩展
   - 调试配置
   - 任务配置

#### 预期收益

- 首次环境搭建时间缩短至 15 分钟
