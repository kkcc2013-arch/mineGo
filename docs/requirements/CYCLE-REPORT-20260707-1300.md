## 1. 新增需求

### REQ-00485: 用户数据批量导出安全管控与审计系统

**类别**：反作弊/安全加固  
**优先级**：P1  
**状态**：new  

**核心功能**：
- 用户导出频率限制（每月2次）
- 管理员批量导出审批工作流（1-3人审批）
- 敏感数据脱敏引擎（邮箱、电话、GPS等）
- 全链路审计日志
- 异常导出检测算法（短时间多次、异常时段、大批量）
- 异步导出任务队列
- 导出文件加密存储与自动过期

**解决痛点**：
1. 缺乏频率限制 - 攻击者可高频请求导出消耗资源
2. 无批量导出防护 - 管理员可无限制导出大量数据
3. 缺少审计追踪 - 无法追溯滥用行为
4. 敏感数据未脱敏 - 违反GDPR最小化原则
5. 无异常行为检测 - 无法识别批量数据窃取

**技术方案**：
- `ExportRateLimiter`：多层限流（用户/管理员/系统级）
- `ExportApprovalWorkflow`：分级审批流程（小/中/大批量）
- `DataMaskingEngine`：智能脱敏（邮箱、电话、GPS、支付信息）
- `ExportAnomalyDetector`：异常检测（Z-score算法）
- `ExportTaskQueue`：异步处理（Bull队列）
