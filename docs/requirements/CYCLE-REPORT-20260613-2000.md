## 📄 新需求详情

### REQ-00167：游戏内容本地化数据层与动态翻译系统

**类别**：国际化/本地化  
**优先级**：P1  
**状态**：done

**背景问题**：
- 项目已有 UI 国际化，但游戏核心内容（精灵、技能、道具）缺少本地化
- 数据库 `pokemon_species` 表仅支持中英文，缺少日语
- 活动公告无法按用户语言动态展示

**解决方案**：
- 扩展数据库表支持中/英/日三语
- 创建 `content_localizations` 通用翻译表
- 实现 ContentLocalizer 服务类
- 提供本地化 API 端点
