### 需求详情
- **类别**: 国际化/本地化
- **优先级**: P2
- **状态**: new
- **涉及服务**: gateway、user-service、reward-service、backend/shared/i18n、admin-dashboard、database/migrations

### 背景
当前游戏内富文本内容（邮件、公告、推送通知、帮助文档）缺少系统性的本地化管理，存在以下问题：
1. 富文本内容本地化覆盖率仅 30%
2. 缺少版本管理和回滚机制
3. 翻译一致性差，无术语库支持
4. 运营效率低，内容发布周期长

### 目标
建立游戏内富文本内容本地化管理平台，实现：
- 富文本内容本地化覆盖率从 30% 提升至 95%
- 内容发布效率提升 70%（从 2 天缩短至 4 小时）
- 翻译一致性错误减少 80%
- 支持实时预览与回滚
