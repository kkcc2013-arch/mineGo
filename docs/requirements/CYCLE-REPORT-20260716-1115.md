```
└── serviceLifecycle.test.js           (11.2 KB) 单元测试
```

**核心模块**:

1. **ServiceLifecycleState** (17 种状态)
   - 启动阶段: uninitialized → starting → waiting_dependencies → ... → healthy
   - 运行阶段: healthy, degraded
   - 关闭阶段: draining → stopping → ... → stopped
   - 异常状态: error

2. **ServiceLifecycleStateMachine**
   - 状态转换验证
   - 事件通知（EventEmitter）
   - 状态历史记录
   - 状态快照导出

3. **GracefulShutdownOrchestrator**
   - 连接排空（Drain）
   - 插件停止
   - 组件关闭（Database/Redis/Kafka/Server）
   - 资源清理

4. **DependencyStartupCoordinator**
   - 并行健康检查
   - 重试机制（2秒间隔）
   - 超时控制

5. **ServiceLifecycleManager**
   - 统一启动/停止接口
   - 组件注册
   - 信号处理（SIGTERM/SIGINT）
   - 状态持久化（Redis）
