## 🔍 REQ-00077 实现详情

### 核心模块
1. **SlowQueryCollector** - 慢查询日志采集
   - 支持 pg_stat_statements 扩展
   - 备用方案：pg_stat_activity
   - 定时采集间隔：5 分钟
   - 阈值：1000ms

2. **QueryPlanAnalyzer** - 执行计划分析
   - 检测扫描类型：Seq Scan、Index Scan、Bitmap Index Scan
   - 生成优化建议：索引建议、分页建议、缓存建议
   - 警告级别：high、medium、low

3. **IndexUsageAnalyzer** - 索引使用率分析
   - 统计索引扫描次数
   - 识别重复索引
   - 排除约束索引（主键、唯一约束）

### API 端点
- `GET /api/slow-query/stats` - 统计信息
- `GET /api/slow-query/top` - Top N 慢查询
- `POST /api/slow-query/analyze` - 执行计划分析
- `GET /api/slow-query/indexes` - 索引使用情况
- `GET /api/slow-query/report` - 完整报告

### Prometheus 指标
- `minego_database_slow_query_total` - 慢查询计数
- `minego_database_slow_query_duration_ms` - 查询延迟直方图
- `minego_database_slow_query_count` - 慢查询总数
- `minego_database_index_usage_ratio` - 索引使用率
- `minego_database_query_execution_time_ms` - 查询执行时间
- `minego_database_buffer_cache_hit_ratio` - 缓存命中率

### 告警规则（8条）
1. HighSlowQueryRate - 慢查询率告警
2. VerySlowQuery - 极慢查询告警（>10s）
3. SlowQuerySpike - 慢查询激增告警
4. LowIndexUsageRatio - 索引使用率低告警
5. HighQueryExecutionTime - P95 执行时间过高
6. LowBufferCacheHitRatio - 缓存命中率低告警
7. SlowQueryTrendIncreasing - 慢查询趋势上升
8. IndexBloatDetected - 索引膨胀检测
