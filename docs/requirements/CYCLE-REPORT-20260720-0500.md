## 1. 新需求生成

### REQ-00604：游戏客户端内存扫描检测与防护系统

**类别**：反作弊（轮转：603 % 18 = 9）
**优先级**：P1
**状态**：new

**背景**：
- 当前已有内存篡改检测（REQ-00163），但缺少对内存扫描工具的主动防护
- GameGuardian、Cheat Engine 等工具可扫描定位关键数值
- 关键数据在内存中以明文存储，易被逆向分析

**目标**：
- 主动检测内存扫描工具运行
- 关键数值动态加密与虚拟化
- 进程保护加固（反调试、反 ptrace）
- 行为风控联动

**预期收益**：
- 减少 95% 以上内存扫描类作弊成功率
- 增加 10 倍逆向分析成本
- 检测延迟从分钟级降至秒级

   - HTTP 方法方差、错误率、敏感 API 命中率
   - 请求间隔统计（均值、标准差、偏度）
   - 会话年龄、认证尝试次数

   - 4 级威胁等级：normal/suspicious/threat/critical
   - 支持自定义规则添加/移除

3. **自动响应**（12 种动作）
   - 日志增强、动态限流、激进限流
   - 验证码挑战（软/硬）
   - 会话标记、会话撤销
   - 告警通知、告警升级
   - IP 临时封禁、请求阻止
