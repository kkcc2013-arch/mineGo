## 一、新需求生成

### REQ-00236：精灵变异系统与稀有形态

| 字段 | 值 |
|------|-----|
| 编号 | REQ-00236 |
| 类别 | 功能增强（轮转序号 0） |
| 优先级 | P1 |
| 状态 | new |
| 涉及服务 | pokemon-service、catch-service、location-service、gateway、game-client |

#### 核心功能
- **7 种变异类型**：闪光、巨大、迷你、极光、暗影、光辉、星辰
- **稀有度分层**：
  - 普通：1/500 概率，+5% 属性加成
  - 稀有：1/2000 概率，+10% 属性加成
  - 传说：1/10000 概率，+15% 属性加成
- **变异图鉴系统**：记录玩家发现的所有变异
- **活动系统**：活动期间概率提升
- **专属视觉效果**：粒子特效、动画、音效

#### P1 优先级理由
1. 核心玩法增强，显著提升用户留存
2. 社交传播价值，有利于口碑传播
3. 商业化潜力，可配合付费道具
4. 竞品对标成功（Pokémon GO 闪光系统）

## 二、已实现需求审核

### REQ-00122：微服务配置中心与动态配置热更新系统 ✅

#### 实现验证

| 组件 | 状态 | 说明 |
|------|------|------|
| ConfigCenter.js | ✅ | 约 450 行核心模块 |
| configRoutes.js | ✅ | 10 个管理 API 端点 |
| 数据库迁移 | ✅ | config_audit_log 表 |
| 热更新机制 | ✅ | Redis Pub/Sub + 本地监听器 |
| 版本控制 | ✅ | 版本号 + 历史记录（100 版本） |
| 降级保护 | ✅ | 默认配置 + 缓存 |

#### 管理 API 端点
- GET /admin/config - 配置概览
- GET /admin/config/:serviceName - 服务配置
- PUT /admin/config/:serviceName/:key - 更新配置
- POST /admin/config/:serviceName/batch - 批量更新
- DELETE /admin/config/:serviceName/:key - 删除配置
- GET /admin/config/:serviceName/history - 变更历史
- POST /admin/config/:serviceName/rollback - 版本回滚
- GET /admin/config/:serviceName/audit - 审计日志
- GET /admin/config/health - 健康检查
