# REQ-00207：精灵对比工具与属性分析系统

- **编号**：REQ-00207
- **类别**：前端体验
- **优先级**：P1
- **状态**：new
- **涉及服务/模块**：game-client、frontend/game-client/src/components/PokemonCompare.js、frontend/game-client/src/components/ComparePanel.css、pokemon-service、gateway
- **创建时间**：2026-06-14 19:00
- **依赖需求**：REQ-00027（精灵详情页 3D 模型展示）

## 1. 背景与问题

当前游戏客户端缺少精灵对比功能。玩家在以下场景面临决策困难：

1. **培养决策**：玩家捕获多只同种精灵后，无法快速对比 IV 值、CP 潜力，需要逐只查看详情页
2. **战斗选择**：准备道馆战斗时，无法直观对比多只精灵的属性克制关系和战斗能力
3. **交易评估**：好友发起交易请求时，无法对比双方精灵价值是否对等
4. **进化选择**：进化前无法对比进化后各形态的属性差异

现有实现需要玩家反复切换精灵详情页，用户体验差，决策效率低。

## 2. 目标

提供一个直观的精灵对比工具，支持：
- 2-4 只精灵并排对比
- 属性、技能、IV、CP 潜力可视化对比
- 战斗能力评分对比
- 进化路径对比

预期收益：玩家决策时间减少 60%，培养满意度提升。

## 3. 范围

- **包含**：
  - 精灵对比 UI 组件（PokemonCompare.js）
  - 属性雷达图对比
  - IV/CP 柱状图对比
  - 技能列表对比
  - 战斗能力评分
  - 快速选择器（从背包选择）
  
- **不包含**：
  - 自动推荐最优精灵（属于 REQ-00076 成就系统）
  - 批量对比超过 4 只精灵
  - 跨账号精灵对比

### 4.2 核心功能

1. **精灵选择器**
   - 从玩家背包选择精灵
   - 支持搜索、筛选
   - 显示精灵缩略图和基本信息

2. **属性雷达图**
   - 使用 SVG 绘制五维雷达图
   - 维度：攻击、防御、体力、速度、潜力
   - 不同精灵用不同颜色区分

3. **IV 对比柱状图**
   - 水平柱状图对比各项 IV
   - 高亮最高/最低值
   - 显示 IV 总分和百分比

4. **战斗能力评分**
   - 综合评分算法：`(攻击×0.4 + 防御×0.3 + 体力×0.3) × IV系数`
   - 道馆进攻评分：考虑属性克制和技能 DPS
   - 道馆防守评分：考虑耐久度和技能覆盖

5. **技能对比**
   - 快速技能 DPS 对比
   - 蓄能技能 DPS 对比
   - 技能类型覆盖分析
