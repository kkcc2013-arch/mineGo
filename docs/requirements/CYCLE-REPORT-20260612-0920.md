## 阶段 1: 新需求生成

### REQ-00151: 精灵羁绊技能解锁机制

- **类别**: 功能增强
- **优先级**: P1
- **状态**: new
- **涉及服务**: pokemon-service、game-client、database/migrations

#### 背景与价值
REQ-00067 已实现精灵羁绊与互动养成系统，但缺少羁绊技能解锁机制，导致：
- 羁绊系统激励不足
- 玩家培养动力不足
- 长期留存率未达预期

#### 关键特性
1. **技能槽位系统**: 基于亲密度等级解锁 3 个技能槽
   - 友好（20-49）→ 第 1 槽
   - 亲密（50-89）→ 第 2 槽
   - 牵绊（90-100）→ 第 3 槽

2. **羁绊技能定义**: 每个精灵类型可拥有 1-3 个专属技能
   - 羁绊电击：亲密度越高，威力越大
   - 守护闪电：为队友提供护盾
   - 十万伏特·羁绊：无视对手抗性

3. **数据库设计**: 2 个新表
   - `bond_skill_definitions`: 羁绊技能定义表
   - `pokemon_bond_skills`: 精灵羁绊技能学习表

4. **API 端点**: 6 个新端点
   - GET /api/pokemon/:id/bond-skills
   - POST /api/pokemon/:id/bond-skills/:skillId/learn
   - DELETE /api/pokemon/:id/bond-skills/:skillId
   - POST /api/pokemon/:id/bond-skills/:skillId/activate
   - GET /api/pokemon-species/:speciesId/bond-skills/available
