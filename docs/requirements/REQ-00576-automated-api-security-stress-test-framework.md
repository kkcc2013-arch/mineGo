## 需求描述

为了确保 API 在高负载情况下的安全稳定性，需要构建一个自动化的安全压力测试框架。该框架应模拟真实业务流量，同时执行安全攻击注入（如 SQL 注入、XSS 注入、慢速攻击），评估 API 在压力下的安全响应性能。

## 技术方案

### 1. 安全注入模块
- 集成 Fuzzing 工具，针对 HTTP 参数、Headers 进行变异注入。
- 支持定义各类安全攻击模板（SQLi, XSS, Command Injection）。

### 2. 负载生成模块
- 使用 k6 或类似的高并发流量生成器，在负载高峰期同步发起安全 payload 请求。

### 3. 监控与分析
- 对比正常负载下的错误率和延迟变化。
- 自动分析安全触发的错误率是否符合预期（如被 WAF/Gateway 阻断）。

## 验收标准

- [ ] 框架能够生成稳定压力负载
- [ ] 能够成功注入安全攻击并记录响应结果
- [ ] 具备安全攻击触发的错误告警功能
- [ ] 能够输出压力下 API 的性能安全影响报告
