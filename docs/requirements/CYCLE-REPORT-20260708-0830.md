### REQ-00499：事件驱动服务编排与分布式状态机引擎

**类别**：可扩展性/解耦（类别轮转：499 % 18 = 13）

**优先级**：P1

**背景**：
mineGo 项目中存在多个跨服务的复杂业务流程（捕捉、Raid 战斗、交易、支付），当前实现分散在各个服务中，存在紧耦合、状态管理分散、异常处理不完整等问题。

**目标**：
