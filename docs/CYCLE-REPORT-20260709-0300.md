## 一、生成新需求

### REQ-00522：数据保留政策透明化与用户通知系统

**类别**：合规/隐私  
**优先级**：P1  
**状态**：new

#### 需求背景
根据 GDPR 第 13、14 条规定，数据控制者必须告知数据主体个人数据的存储期限和处理方式。当前项目虽然已实现 DataLifecycleManager，但缺少面向用户的保留政策透明展示和到期通知机制。

#### 核心功能
1. **保留政策 API**：查询各类数据的保留期限和处理方式
2. **隐私中心页面**：用户可查看保留政策、数据状态、即将到期的数据
3. **通知系统**：邮件、推送、站内信多渠道通知（到期前 30/7/1 天）
4. **数据状态追踪**：用户个人数据保留时间线可视化
5. **管理员配置**：动态调整保留政策和通知时机
6. **多语言支持**：中英文保留政策展示

#### 工作量估算
L（Large），约 8 人天
