#### 核心功能
1. **数据聚合层**
   - 从 Prometheus 查询错误率、延迟、吞吐量指标
   - 从 Jaeger 获取服务拓扑和依赖关系
   - 定时刷新机制（默认 5 秒）
   - 核心链路 SLA/SLO 监测（注册、登录、捕捉、对战）

2. **智能告警系统**
   - 阈值监控：错误率 5%、延迟 500ms、超时率 1%
   - 告警分级：critical / high / medium / low
   - 告警去重：5 分钟内相同类型不重复
   - 告警确认机制

3. **实时推送**
   - WebSocket 毫秒级数据更新
   - 多客户端支持
   - 自动重连机制
   - 健康检查端点

4. **可视化大屏**
   - SLA/SLO 指标卡片（实时延迟、错误率、吞吐量）
   - 服务拓扑图（Canvas 绘制，健康状态高亮）
   - 告警列表（分级显示，支持确认）
   - 趋势图表（Chart.js 错误率、延迟趋势）
   - 响应式设计
