## 5. 性能预估

### 5.1 翻译延迟
- **缓存命中**：< 50ms
- **API 调用**：200-500ms（取决于翻译引擎）
- **P95 预估**：< 500ms ✅

### 5.2 缓存命中率
- **Redis 缓存**：热数据命中率 > 80%
- **数据库缓存**：冷数据命中率 > 40%
- **综合命中率预估**：> 60% ✅

### 5.3 成本估算
- **每日翻译量**：50 万条消息
- **缓存命中后 API 调用**：约 20 万次/天
- **字符数**：约 1000 万字符/天
- **成本（DeepL）**：约 $20-50/天
