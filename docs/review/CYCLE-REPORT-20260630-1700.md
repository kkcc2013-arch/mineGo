## 1. 生成新需求（类别：技术债/重构）

### REQ-00391：console.log 替换与结构化日志强制使用系统

**编号**: REQ-00391
**类别**: 技术债/重构（类别 12，轮转序号 = 390 % 18 = 12）
**优先级**: P1
**状态**: new

**背景问题**:
- backend/shared 目录 190 个模块中存在 **62 处** console.log/console.error/console.warn 调用
- 项目已有完善的 pino 结构化日志系统（logger.js）
- console 调用无法关联 traceId，导致分布式追踪断裂
- 生产环境日志无法聚合分析

**解决方案**:
1. 消除所有 backend/shared 模块中的 console 调用
2. 添加 ESLint `no-console` 规则阻止新 console 引入
3. 建立 CI/CD lint 检查规则
4. 提供迁移指南和最佳实践文档

**涉及模块**: backend/shared、所有微服务、gateway、CI/CD
