### 关键特性

#### 1. 三种关联方式
- **拓扑关联**：基于服务依赖图关联上下游告警
- **时间关联**：基于时间窗口关联同时段告警
- **因果关联**：基于预定义规则关联因果告警

#### 2. 根因推断算法
- 严重程度权重计算
- 因果关系得分
- 拓扑位置得分（上游更可能是根因）
- 时间得分（更早的告警更可能是根因）

#### 3. 智能降噪
- 抖动检测（Flapping）
- 低优先级抑制
- 历史模式匹配

#### 4. 服务拓扑
