## 4. 遗留问题与改进建议

### 4.1 遗留问题

1. **前端虚拟滚动未集成**
   - `game-client` 精灵列表组件需要实现虚拟滚动
   - 需要在前端任务中单独实现
   - 影响：用户在查看大量精灵时可能仍感卡顿

2. **拼音搜索未完全实现**
   - 当前仅支持精灵名称模糊匹配
   - 中文用户期望拼音搜索（如搜索 "pi" 匹配 "皮卡丘"）
   - 需要额外集成拼音索引库

### 4.2 改进建议

1. **增加拼音索引**
   - 集成 `pinyin` npm 包
   - 在精灵数据中增加 `pinyin` 字段
   - 支持拼音首字母搜索

2. **实现前端虚拟滚动**
   - 在 `game-client` 中实现 `VirtualPokemonList` 组件
   - 支持滚动预加载
   - 提升大量精灵渲染流畅度

3. **增加搜索建议**
   - 实现搜索词自动补全
   - 提供热门搜索词推荐
   - 提升搜索体验

4. **优化缓存 TTL**
   - 根据用户活跃度动态调整缓存时长
   - 高频用户延长缓存，低频用户缩短缓存
   - 平衡内存占用与响应速度
