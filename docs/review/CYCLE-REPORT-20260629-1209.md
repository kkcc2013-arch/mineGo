## ✅ 实现需求

### REQ-00348: 精灵背包智能整理与自动分类系统

| 字段 | 值 |
|------|-----|
| 编号 | REQ-00348 |
| 类别 | 功能增强 |
| 优先级 | P1 |
| 状态 | done |

**核心模块**:

1. **InventorySorter** - 多维度排序引擎
   - 9种排序维度: 战斗力、CP、时间、类型、稀有度、亲密度、进化潜力、IV总和、等级
   - 5种分组方式: 类型、用途、稀有度、世代、收藏
   - 过滤功能: 类型、CP范围、稀有度、收藏/锁定、搜索、自定义标签

2. **OrganizationAdvisor** - 智能整理建议服务
   - 重复精灵识别与建议转移
   - 低价值精灵识别（CP < 500）
   - 战斗队伍推荐（考虑类型多样性）
   - 背包容量预警（80%/95%阈值）
   - 快速操作建议生成

3. **API 路由** - 12 个端点
   - GET /inventory - 精灵列表（排序/分组/过滤）
   - GET /inventory/advice - 智能整理建议
   - POST /inventory/favorite - 收藏操作
   - POST /inventory/lock - 锁定操作
   - POST /inventory/batch-transfer - 批量转移
   - 等...

4. **数据库迁移**
   - 新增字段: is_favorite, is_locked, custom_tags, nickname
   - 用户背包偏好表
   - 用户糖果表（转移奖励）
   - 战斗队伍表
   - 多个优化索引
