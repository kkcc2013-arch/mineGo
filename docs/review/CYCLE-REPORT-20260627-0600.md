- 自定义指标适配器配置文件

- 可配置参数：预测窗口、提前量、最低置信度

#### 5. 扩缩容 API
- `/api/v1/autoscaling/status` - 引擎状态
- `/api/v1/autoscaling/predictions` - 扩容建议
- `/api/v1/autoscaling/execute` - 手动执行
- `/api/v1/autoscaling/services/:name/prediction` - 单服务预测
- `/api/v1/autoscaling/efficiency` - 资源效率报告

#### 6. 扩缩容指标（20+）
