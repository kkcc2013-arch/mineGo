#### 1. 核心模块

**SpawnEngine.js** (backend/shared/SpawnEngine.js)
- 基于热力图的动态刷新算法
- 时间因子：高峰时段（18-21点）刷新密度增加 50%+
- 玩家活跃度因子：玩家越多刷新越多（上限 1.6 倍）
- 事件因子：社区日 2 倍、聚焦时刻 1.5 倍
- 加权随机选择精灵池
- 手动刷新功能（运营工具）

**HeatmapCollector.js** (backend/shared/HeatmapCollector.js)
- 实时追踪各区域活跃玩家数
- 使用 Redis sorted set 存储活跃玩家（自动过期）
- 每小时统计数据写入数据库
- 峰值时段预测功能

**spawnMetrics.js** (backend/shared/spawnMetrics.js)
- Prometheus 指标暴露：
  - spawn_active_total：当前活跃刷新数量
  - spawn_created_total：刷新计数器
  - spawn_despawn_total：消失计数器
  - spawn_capture_rate：捕捉成功率
  - spawn_cell_active_players：区域热度

#### 2. API 接口

**spawnConfig.js** (backend/services/location-service/src/routes/spawnConfig.js)
- GET /api/v1/spawn/config/cell/:geohash - 获取区域配置
- PUT /api/v1/spawn/config/cell/:geohash - 更新区域配置
- POST /api/v1/spawn/config/cells/batch - 批量更新区域配置
- POST /api/v1/spawn/events - 创建活动事件
- GET /api/v1/spawn/events - 获取活动列表
- PUT /api/v1/spawn/events/:id - 更新活动事件
- DELETE /api/v1/spawn/events/:id - 删除活动事件
- GET /api/v1/spawn/pool/:biome - 获取精灵池配置
- PUT /api/v1/spawn/pool/:biome - 更新精灵池
- POST /api/v1/spawn/manual - 手动刷新精灵
