## 1. 新需求生成

### REQ-00593：灾备故障场景自动发现与混沌验证覆盖系统

| 项目 | 内容 |
|------|------|
| 编号 | REQ-00593 |
| 类别 | 容灾/高可用 |
| 优先级 | P1 |
| 状态 | new |
| 创建时间 | 2026-07-19 03:00 |

**核心功能**：
- 服务拓扑分析器：解析 K8s Deployment、Docker Compose 配置
- 依赖图构建器：分析 HTTP/RPC/Kafka/Redis 调用链
- 故障场景生成器：自动生成单点故障、级联故障、网络分区等场景
- 风险评分引擎：计算场景风险分数（影响面 × 概率 × 恢复难度）
- 混沌实验映射器：将场景转换为 ChaosEngine 可执行实验
- 覆盖率计算器：量化灾备能力覆盖度，识别验证盲区

**解决的问题**：
- 当前故障场景依赖人工枚举，难以覆盖所有潜在风险
- 系统无法自动识别单点故障（SPOF）、依赖链脆弱点
- 灾备切换能力验证覆盖率未知
