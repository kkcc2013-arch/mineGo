### 核心功能

**告警级别**：
- 🔴 **Critical**：熔断器打开 → 全渠道立即通知
- 🟠 **Warning**：熔断器半开 → Webhook + 日志
- 🟢 **Info**：熔断器关闭 → 仅日志

**告警聚合**：
- 30 秒内同类型告警自动合并
- 避免告警风暴，减少噪音

**API 接口**：
- `GET /api/admin/alerts/history` - 查询告警历史
- `GET /api/admin/alerts/silences` - 查询静默规则
- `POST /api/admin/alerts/silences` - 创建静默规则
- `DELETE /api/admin/alerts/silences/:pattern` - 删除静默规则
- `POST /api/admin/alerts/test` - 发送测试告警
