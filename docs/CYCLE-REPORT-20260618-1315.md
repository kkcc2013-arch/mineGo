### 核心功能

#### 1. SQL 注入检测

```javascript
// 检测模式
- OR/AND 注入: /(\bOR\b|\bAND\b)\s*['"]?\d+['"]?\s*=\s*['"]?\d+['"]?/i
- UNION 注入: /UNION\s+(ALL\s+)?SELECT/i
- 堆叠查询: /;\s*(DROP|DELETE|UPDATE|INSERT|EXEC|TRUNCATE|ALTER)/i
- 注释截断: /--\s*$|\/\*.*\*\//
- 时间盲注: /WAITFOR\s+DELAY|SLEEP\s*\(|BENCHMARK\s*\(/i
```

#### 2. NoSQL 注入检测

```javascript
// 检测模式
- $where 注入: /\$where/i
- 原型污染: /__proto__|constructor|prototype/i
- Redis 键名白名单验证
```

#### 3. XSS 防护

```javascript
// 多上下文编码
- HTML 实体编码: & < > " ' ` / =
- HTML 属性编码
- JavaScript 编码
- URL 编码
- CSS 编码
```

#### 4. 路径遍历检测

```javascript
// 检测模式
- 路径遍历: /../
- URL 编码: /%2e%2e/i
- 路径规范化验证
```

#### 5. 命令注入检测

```javascript
// 检测模式
- 命令链接: /;\s*(ls|cat|rm|wget|curl|bash)/i
- 管道注入: /\|\s*(ls|cat|rm|wget|curl)/i
- 反引号执行: /`[^`]+`/
```
