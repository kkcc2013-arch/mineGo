#### Prometheus 指标

| 指标 | 类型 | 说明 |
|------|------|------|
| minego_slo_target | Gauge | SLO 目标值 |
| minego_slo_budget_total | Gauge | 总错误预算 |
| minego_slo_budget_remaining | Gauge | 剩余预算 |
| minego_slo_budget_remaining_ratio | Gauge | 剩余比例 |
| minego_slo_burn_rate | Gauge | 燃尽率（多周期） |
| minego_slo_circuit_state | Gauge | 熔断器状态 |
| minego_slo_degradation_level | Gauge | 降级级别 |

- P0: 预算耗尽（< 2%）→ 自动降级

- P1: 预算严重不足（< 5%）→ 只读模式
- P1: 预算将在 24h 内耗尽 → 预测告警

- 燃尽率趋势图（1h/24h）
