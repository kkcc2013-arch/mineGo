### REQ-00434: WebSocket 消息完整性与防重放攻击保护系统

- **类别**: 安全加固
- **优先级**: P1
- **状态**: new
- **涉及模块**: gateway、backend/shared/WebSocketSecurity.js

**背景**: 当前 WebSocket 消息缺少签名和防重放保护，存在中间人攻击风险

**核心功能**:

- 异常行为检测与自动断开
