## 技术亮点

### 死锁分析能力

1. **多维度解析**
   - 进程 ID 提取
   - 表 OID 关联
   - 锁类型识别（ShareLock、ExclusiveLock 等）

2. **严重性评估**
   - LOW: 单次死锁，自动重试成功
   - MEDIUM: 短时间内多次死锁
   - HIGH: 高频死锁
   - CRITICAL: 死锁导致事务失败

3. **模式分析**
   - 热点表统计
   - 操作分布分析
   - 时间分布统计
   - 服务分布统计
