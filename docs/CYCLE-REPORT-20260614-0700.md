**API 端点**：
- `GET /api/pokemon/backup/list` - 获取备份列表
- `POST /api/pokemon/backup/create` - 创建手动备份
- `POST /api/pokemon/backup/restore/:id` - 从备份恢复
- `DELETE /api/pokemon/backup/:id` - 删除备份
- `GET /api/pokemon/backup/export` - GDPR数据导出
- `POST /api/pokemon/backup/auto-backup` - 设置自动备份
- `GET /api/pokemon/backup/quota` - 获取配额信息

**技术亮点**：
- AES-256-GCM 加密存储
- gzip 压缩减少存储空间
- 三种恢复模式：merge/replace/append
- 三种冲突处理策略
- 每日/每周自动备份
- 过期备份自动清理
