## 📚 使用技巧

### 1. 提示词优化建议

**基础结构**:
```
[主体描述] + [动作/状态] + [环境场景] + [光线照明] + [摄影技术] + [美学风格] + [技术参数]
```

**关键要素**:
- **主体**: 明确人物年龄、性别、外貌特征
- **动作**: 具体描述运动类型和速度
- **环境**: 场景细节和氛围
- **光线**: 光源类型和色温
- **摄影**: 镜头类型、角度、帧率
- **风格**: 艺术风格和美学方向

### 2. 不同模型特性

**Runway Gen-3**:
- 优势: 电影级质量、人物动作流畅、复杂场景处理强
- 适用: 高端商业内容、艺术项目
- 建议: 详细描述光线和构图

**OpenAI Sora**:
- 优势: 长视频生成、复杂叙事、物理真实
- 适用: 剧情短片、叙事内容
- 建议: 提供完整故事背景和连续动作

**Pika Labs**:
- 优势: 风格化效果好、速度快、易用性强
- 适用: 社交媒体内容、快速制作
- 建议: 简洁清晰的提示词

**Kling AI**:
- 优势: 人物动作自然、亚洲面孔表现佳
- 适用: 亚洲市场内容、人物特写
- 建议: 强调动作流畅性和表情细节

**Hailuo AI**:
- 优势: 艺术风格多样、创意表现力强
- 适用: 创意项目、艺术内容
- 建议: 明确艺术风格参考

### 3. 帧率选择指南

- **24fps**: 电影质感、叙事内容
- **30fps**: 自然流畅、日常内容
- **60fps**: 流畅运动、运动场景
- **120fps+**: 慢动作效果、细节展示

### 4. 镜头语言

- **Close-up (特写)**: 情感表达、细节展示
- **Medium shot (中景)**: 人物互动、上半身动作
- **Full shot (全景)**: 全身展示、环境关系
- **Extreme close-up (极致特写)**: 眼睛、微表情
- **Wide shot (广角)**: 场景氛围、环境叙事

### 5. 光线描述

- **Golden hour (黄金时刻)**: 日出日落、温暖柔和
- **Blue hour (蓝调时刻)**: 黄昏、冷色调浪漫
- **Natural light (自然光)**: 真实、纪录片感
- **Studio lighting (摄影棚光)**: 专业、可控
- **Backlighting (逆光)**: 轮廓光、剪影效果
- **Soft diffused light (柔光)**: 美化、梦幻

### 6. 常见问题解决

**人物面部变形**:
- 添加 "detailed facial features, consistent face"
- 使用 "high resolution, 4K quality"

**动作不自然**:
- 明确帧率 "30fps natural motion"
- 添加 "smooth movement, realistic physics"

**风格不明确**:
- 参考具体作品 "inspired by [film/director]"
- 使用多个风格标签

**背景杂乱**:
- 添加 "shallow depth of field, bokeh background"
- 使用 "clean composition, negative space"
